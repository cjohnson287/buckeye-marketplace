# Buckeye Marketplace User & Admin Guide

## User Guide

### 1. Browsing Products
1. Open the production frontend: https://buckeye-frontend-final-ryon287.azurewebsites.net
2. View the product listings on the homepage.
3. Review product cards for title, description, price, category, seller, and image.

### 2. Adding Items to Cart
1. Choose a product from the product list.
2. Click the add-to-cart option.
3. Open the cart page to confirm the item appears.
4. Review item quantity and total price.

### 3. Creating an Account and Logging In
1. Navigate to the account, register, or login area.
2. Create a new account using valid information.
3. Log in with the account credentials.
4. Confirm that user-specific features become available.

### 4. Placing an Order
1. Add one or more items to the cart.
2. Go to checkout.
3. Review the order details.
4. Submit/place the order.
5. Confirm the order is created successfully.

### 5. Viewing Order History
1. Log in as a user.
2. Navigate to order history or account orders.
3. Review previous orders and order details.

## Admin Guide

### 1. Managing Products
1. Log in with an admin account.
2. Navigate to the admin/product management area.
3. Add, edit, or remove products as needed.
4. Confirm product changes appear in the product listing.

### 2. Managing Orders
1. Log in with an admin account.
2. Navigate to the admin/order management area.
3. View submitted orders.
4. Update order status as needed.
5. Confirm order changes save correctly.
